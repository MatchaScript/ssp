# @matchalatte/ssp-color-editor

Color editor for ssp.

## Usage

```sh
npx @matchalatte/ssp-color-editor
```

Starts a local static server and opens the editor in your browser.

### Options

```
-p, --port <port>   Port to listen on (default: 4321, env PORT)
    --host <host>   Host to bind (default: 127.0.0.1)
    --no-open       Do not open the browser automatically
-h, --help          Show help
```
